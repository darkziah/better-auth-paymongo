## How It Works

The Autumn pattern is a no-webhook billing architecture:

<Steps>
  <Step title="Attach">
    Create a checkout session and redirect user to PayMongo
  </Step>
  <Step title="Check">
    When user accesses a feature, verify payment status via API
  </Step>
  <Step title="Track">
    Decrement usage for metered features
  </Step>
</Steps>
